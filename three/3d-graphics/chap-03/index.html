# Three.js 의 다양한 광원 (Light)

광원이 없다면 Object가 제대로 렌더링 되는 것을 볼 수 없습니다.
Three.js 에서 사용하는 광원에 대해 알아보겠습니다.

TIP

WebGL 자체는 광원을 지원하지 않습니다. Three.js 가 없다면 이런 광원을 흉내낼 수 있는 셰이더 프로그램을 작성해야 합니다.

# 1. AmbientLight

  • 가장 기본으로 Scene의 모든 객체들에게 균일하게 조명을 제공합니다.
  • 이 광원은 방향이 없기 때문에, 그림자를 만들지는 못합니다.
  • 일반적으로 단독으론 이용하지 않으며, SpotLight나 DirectionalLight와 함께 사용하며 그림자를 부드럽게 하거나 추가 색상을 더하는데 사용합니다.
  • PointLight와 SpotLight, DirectionalLight 는 아래 그림과 같이 빛을 방출합니다.

# 2. PointLight

  • PointLight 는 특정 지점에서 모든 방향으로 빛을 방출합니다.
  • 밤하늘에 쏘아 올린 조명탄과 비슷하지요.
  • 아래 예제에서 PointLight 의 색상과 Visibility 를 조절할 수 있습니다.

# 3. SpotLight

  • 원뿔 모양의 효과를 가지는 이 광원도 자주 사용됩니다.
  • 플래시 라이트나 랜턴과 유사합니다.
  • 이 광원은 빛이 만들어진 곳에서 방향성과 각도를 가집니다.

# 4. DirectionalLight

  • 이 광원은 아주 멀리까지 도달할 수 있습니다.
  • 이 빛이 보내는 광선들은 서로 나란하게 나아갑니다. (태양 광선)
  • SpotLight 와 다른 점은 거리에 따라 빛이 소멸되지 않는 것입니다.
  • 이 광원에 의해 조명되는 영역은 동일한 빛의 세기를 가집니다.
  • 아래 예제에서 target을 지정해볼 수 있습니다.

# 5. THREE.HemisphereLight

  • 자연스러운 야외 조명을 만들때 사용합니다.
  • 야외에 있을 때, 빛은 대기에 의해 확산되고 지면과 다른 물체에 반사됩니다.
  • 이 광원은 위와 같은 상황을 위해 만들어졌습니다.
속성 설명
groundColor 지면에서 방출되는 색상
color 하늘에서 방출되는 색상
intensity 빛이 비추는 강도
